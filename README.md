# 👕 Animated T-Shirt Order Button

A creative and interactive **T-Shirt Order Button** built with HTML, CSS, JavaScript, GSAP, and Splitting.js. The project features a unique T-shirt cannon animation with smooth text transitions and engaging micro-interactions.

## ✨ Features

- Interactive T-shirt cannon animation
- Smooth GSAP animations
- Animated text transitions
- Responsive design
- Lightweight and reusable
- Easy to customize
- Modern UI

## 📂 Project Structure

```text
Animated-T-Shirt-Order-Button/
│── index.html
│── style.css
│── script.js
└── README.md
```

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/Animated-T-Shirt-Order-Button.git
```

2. Open `index.html` in your browser.
